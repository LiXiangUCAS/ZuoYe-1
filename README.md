## 安装步骤

```
$ git clone git@github.com:LiXiangUCAS/ZuoYe-1.git
$ cd ZuoYe-1
$ bundle install
$ bundle exec rake db:test:prepare
$ bundle exec rake db:migrate
$ rails server
```
